📁 Comment ça fonctionne ?

Ce fichier HTML est autonome : téléchargez-le, ouvrez-le dans votre navigateur, et votre casier personnel est créé. Vos ressources sont sauvegardées localement. Pour partager vos ressources avec quelqu'un, exportez-les en JSON depuis le panneau admin.

👥 Et les autres utilisateurs ?

Chaque personne qui ouvre ce fichier voit son propre casier vide, complètement séparé du vôtre. Il n'y a pas de serveur commun — impossible de voir les ressources des autres.

📖 Guide d'utilisation — Admin

1

Accéder au mode admin

Cliquez sur ⚙ Admin dans la barre de navigation en haut à droite. Mot de passe par défaut : maths4ever. Un badge vert ✦ Admin apparaît.

2

Ajouter une ressource

Cliquez sur + Ajouter (visible uniquement en mode admin). Remplissez le titre, l'URL principale, les descriptions, une image d'aperçu et des tags.

3

Cadrer une image

Uploadez une image ou collez une URL. Glissez l'image dans le cadre pour la repositionner. Le curseur Zoom permet de l'agrandir ou réduire (10% à 300%). Le cadre représente exactement ce qui sera affiché.

4

Pièces jointes (PDF / HTML)

En bas du formulaire, ajoutez autant de pièces jointes que vous souhaitez (PDF ou fichier HTML local). Elles apparaissent sous la fiche détail comme boutons distincts.

5

Filtres ET

Les filtres sont exclusifs : sélectionner "jeu" ET "maths" n'affiche que les ressources ayant les DEUX tags. Cliquer sur un tag actif le désélectionne.

6

Changer le mot de passe

Dans le panneau admin, section 🔑 Changer le mot de passe. Configurez ensuite les questions secrètes pour pouvoir le récupérer en cas d'oubli.

7

Sauvegarder

Exportez vos données en JSON depuis le panneau admin. C'est votre seule sauvegarde — les données sont stockées dans votre navigateur uniquement.

⚠ Attention : si vous videz les données du navigateur ou changez d'appareil, toutes les ressources sont perdues. Exportez régulièrement.

🔒 À propos des données

✅ Toutes vos données restent sur votre appareil. Aucun serveur. Aucun cookie. Aucun tracker.

Stockage local

Ce portail utilise le localStorage de votre navigateur. Aucune donnée ne transite sur Internet.

Mot de passe

Le mot de passe n'est jamais stocké en clair. Seul son hash SHA-256 est conservé, localement.

Images

Les images uploadées sont stockées en base64 dans le localStorage. Les images volumineuses peuvent alourdir le stockage.

Google Fonts

La police est chargée depuis fonts.googleapis.com. Seule votre IP est transmise à Google (requête HTTP standard).

Suppression

Paramètres navigateur → Confidentialité → Effacer les données de navigation → Cookies et données de sites.
